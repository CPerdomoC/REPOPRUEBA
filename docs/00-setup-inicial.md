# Bloque 1 - Setup inicial del proyecto

## Objetivo

Inicializar una base tecnica estable para desarrollar la app de gestion de libros con enfoque incremental.

## Stack base inicializado

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- ESLint
- Zod
- Radix UI (base con `@radix-ui/react-slot`)

## Estructura creada

- `app/`
- `app/api/` (pendiente de implementar endpoints en el siguiente bloque)
- `features/books/`
- `core/`
- `components/books/`
- `basedatos/`
- `docs/`

## Estado al finalizar este bloque

1. Proyecto inicial generado y funcional.
2. Dependencias base para validacion y componentes UI instaladas.
3. Estructura de carpetas preparada para implementar dominio, API y base de datos.

## Siguientes pasos

1. Definir esquema SQL inicial en `basedatos/`.
2. Crear tipos y validaciones del dominio en `features/books/`.
3. Implementar endpoints en `app/api/libros/`.
4. Construir UI de dashboard y listado en `app/`.
